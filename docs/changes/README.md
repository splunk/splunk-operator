# Change Intent Docs

`docs/changes/` captures doc-first intent for implementation changes.

Use `scripts/dev/start_change.sh "<topic>"` to create a new file.

## Required Sections

Each change-intent file must contain:
- `## Intent`
- `## Scope`
- `## Constitution Impact`
- `## Harness Coverage Plan`
- `## Test Plan`
- `## Runtime Issue Tracker Review`
- `## Implementation Log`
