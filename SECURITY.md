# Security Policy

## Reporting a Vulnerability
If you believe you have found a security vulnerability, please report it
privately. Do **not** open a public GitHub issue.

Contact: `security@splunk.com`

We will acknowledge your report, assess the impact, and provide a timeline for
remediation as soon as possible.

## Supported Versions
Security fixes are applied to currently supported releases. See `docs/ChangeLog.md`
for version history.
